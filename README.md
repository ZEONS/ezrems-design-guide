이지램스(ezREMS) 프로젝트의 웹 표준 템플릿과 디자인 시스템 가이드입니다.

| 파일명 | 용도 |

## 📁 프로젝트 구조 (`ezrems-design-system/`)

표준 임대계약관리 템플릿의 소스 구조입니다.

```text
ezrems-design-system/
├── index.html            (14KB)  # 임대계약관리 표준 템플릿
├── README.md                     # 사용 가이드
├── css/
│   ├── variables.css     (1.3KB) # CSS 변수 (색상/폰트/간격)
│   ├── base.css          (0.9KB) # 리셋 + 기본 스타일
│   ├── layout.css        (4.9KB) # GNB, 탭바, 스플릿 레이아웃
│   └── components.css    (6.0KB) # 버튼, 배지, 폼, 테이블
└── js/
    ├── common.js          (6.7KB) # 공통 유틸리티 (날짜, 숫자, DOM, Toast, Ajax)
    └── page.js            (7.0KB) # 페이지 비즈니스 로직 (샘플 데이터 포함)
```

## ✅ 주요 기능 및 동작 확인

미리보기 화면에서 테스트가 완료된 주요 기능들입니다.

- **GNB 메뉴 + 탭바**: 오리지널과 동일한 구조 및 인터랙션 구현
- **검색 필터 + 조회**: 클릭 시 로딩 스피너 작동 → 결과 표시 → 성공 Toast 알림
- **테이블 인터랙션**: 
  - 행(Row) 클릭 시 파란 배경 하이라이트
  - 왼쪽 파란색 border 강조 적용
- **스마트 배지**:
  - **날짜 배지**: 만료/임박/미래 등 날짜 데이터에 따른 색상 자동 적용
  - **태그 배지**: 쥬라기, 101동 등 구분용 회색 배경 배지
- **파일 제어**: 엑셀 버튼 클릭 시 CSV 파일 자동 생성 및 다운로드
- **데이터 삭제**: 삭제 버튼 클릭 시 `confirm` 확인창 → 확인 시 삭제 Toast 알림
- **편의 기능**: 페이지 하단 스크롤 이동 버튼

## 🚀 새 페이지 개발 가이드

새로운 페이지를 추가하거나 테마를 변경하는 방법입니다.

1. **페이지 추가**: `index.html` 파일을 복사하여 새로운 HTML 파일로 저장합니다.
2. **로직 구현**: `js/page.js` 파일을 복사하여 해당 페이지 전용 데이터/비즈니스 로직을 작성합니다.
3. **테마 변경**: `css/variables.css` 파일 내의 CSS 변수값만 수정하면 전체 테마(Brand Color 등)를 손쉽게 변경할 수 있습니다.
